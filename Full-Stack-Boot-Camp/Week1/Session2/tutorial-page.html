Getting Started

HTML stands for hypertext markup language, it gives meaning and structure to web content. "Hypertext" refers to the links used to connect webpages together. By uploading webpages and linking them together, you become a participant in the world wide web!

Basic HTML Structrure

Any HTML document starts with the special doctype node <!DOCTYPE html> which lets the browser know to use the standards rendering mode. The bulk of an HTML document is made up of content surrounded by element tags. It is important to close your tags properly so elements are properly separated and displayed. The first and most important tag is the <HTML> tag. It surrounds every other element and lets the browser know what language the document is in. It is important to close your tags properly so elements are properly separated and displayed.

The next tag is nested inside of the <HTML> element, and is thus known as a "child" of that element. The <head> tag holds metadata information such as the document title, character set, stylesheet links, etc. Under the <head> element is the <body> element, which is the parent container to the content displayed on a web browser. At the bottom of your document you can use the <footer> element to hold data like a navbar, external links, or other extra information.

Text Formatting

Text formatting helps to make your documents more readable

  • The strong element makes words bold
  • The em element makes words italicised
  • The mark element highlights your text
  • The small element makes words small
  • The code element makes words look like they're code
